Preguntas de descubrimiento para reconstituir el contexto del sistema de Centros Barriales

- ¿Cuál es la pila tecnológica actual (frontend, backend, BD, caché, mensajería, autenticación)?
- ¿Qué flujos críticos de negocio existen (ej. registro/login, gestión de centros, reservas, reportes)?
- ¿Cuál es el modelo de datos principal y las entidades clave?
- ¿Qué APIs exponen contratos (OpenAPI/Swagger) y qué autenticación requieren?
- ¿Qué servicios externos o dependencias existen (p. ej., pagos, notificaciones)?
- ¿Cuáles son los requisitos de rendimiento y SLA objetivo?
- ¿Cómo se maneja la seguridad (roles, permisos, validaciones, cifrado)?
- ¿Qué herramientas se usan para observabilidad (logs, métricas, tracing) y dónde están los dashboards?
- ¿Cómo se despliegan las aplicaciones (CI/CD, entornos, infraestructura)?
- ¿Qué reglas de negocio o cumplimiento regulatorio aplican (protección de datos, GDPR, etc.)?
- ¿Qué archivos clave se pueden compartir para empezar (README, diagramas, contratos API, schema de BD)?
- ¿Qué definitivos se deben evitar compartir (secretos, credenciales, tokens)?

Formato de respuesta sugerido
- Pega aquí resúmenes breves o adjunta archivos clave; si no puedes, responde con respuestas concisas a cada pregunta.
